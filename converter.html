위하고 엑셀 → JSON 변환기

사용법

위하고에서 받은 엑셀(또는 직접 수정한 physical_baseline.xlsx, adjustments.xlsx) 파일을 아래 상자에 드래그하시면 자동으로 JSON으로 변환되어 다운로드됩니다. 변환된 JSON 파일을 카페24 파일업로더에 덮어쓰기로 올리시면 대시보드에 바로 반영됩니다.

판매일(월)보 (SHDV*) → sales.json
현재고 총괄현황 (SHSV*) → inventory.json
미수채권상세현황 (SHBM*) → receivable.json
거래처정보 (거래처관리 내보내기) → clients.json
실재고 기준 (physical_baseline.xlsx) → physical_baseline.json
수동 조정 (adjustments.xlsx) → adjustments.json
파일명 관계없이 내용 자동 인식 · 거래처 파일은 여러 개 한 번에 드래그해도 자동 병합
