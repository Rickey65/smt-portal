데이터 업데이트 방법 (관리자 전용)

매일 작업 (위하고 데이터)
  1. 위하고 SmartA10에서 판매일보 / 현재고 / 미수채권 3개 엑셀 내려받기
  2. 같은 폴더의 converter.html을 브라우저로 열고 → 받은 엑셀 3개를 화면에 드래그
  3. 자동 변환된 sales.json / inventory.json / receivable.json 3개가 다운로드됨
  4. 카페24 → 디자인(PC/모바일) → 파일업로더에서 위 JSON 3개 덮어쓰기 업로드
실사 후 (어쩌다 한 번)
  1. 재고조사 직원이 physical_baseline.xlsx 열어서 기준일자·실재고 갱신 → 저장
  2. 변환기에 드래그 → physical_baseline.json 다운 → 카페24 덮어쓰기
위하고 안 거치는 입출고 발생 시
  1. adjustments.xlsx에 새 행 추가 → 변환기 드래그 → adjustments.json 카페24 덮어쓰기
