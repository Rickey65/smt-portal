# 매일 사용법 (3분)

## 1. 위하고 엑셀 받기

### 서울기연 (3개)
- 판매일보 (SHDV*)
- 현재고 총괄현황 (SHSV*)
- 미수채권 상세현황 (SHBM*)

### 건웅 (3개)
- 똑같이 3종

## 2. 엑셀을 폴더에 떨구기

```
세션 3 SMT서울기연 - 영업/
  └── _auto_sync/
      └── 위하고다운로드/
          ├── 서울기연/  ← 서울기연 엑셀 3개
          └── 건웅/      ← 건웅 엑셀 3개
```

## 3. 바탕화면 "위하고 데이터 동기화" 더블클릭

검은 콘솔창이 자동으로:
- 엑셀 6개를 JSON으로 변환
- GitHub에 자동 업로드
- Vercel이 자동 배포

완료 메시지 → 엔터 → 1~2분 뒤 사이트 새로고침
